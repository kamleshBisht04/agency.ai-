# Agency Landing Page

A modern and responsive **Agency Landing Page** built using **React** and **Tailwind CSS**.
This project is designed to showcase agency services, portfolio, testimonials, and contact information with a clean and professional UI.

## 🚀 Features

* Responsive design (Mobile, Tablet, Desktop)
* Modern UI with Tailwind CSS
* Reusable React components
* Smooth layout for agency presentation
* Portfolio / Projects showcase
* Testimonials section
* Contact section

## ⚙️ Installation

Clone the repository

git clone https://github.com/your-username/agency-landing-page.git

Go to project folder

cd agency-landing-page

Install dependencies

npm install

Start development server

npm run dev
